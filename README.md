# twitter-clone
Backend server for twitter clone with basic features in Node.js, Express.js and MongoDB. <br>
<h2> Features Included : </h2>
<ul>
  <li>Signup/Login </li>
  <li> Tweet</li>
  <li>Follow </li>
  <li> List of all tweets</li>
  <li> List of tweets of following persons only</li>
  <li> Search User</li>
</ul>
